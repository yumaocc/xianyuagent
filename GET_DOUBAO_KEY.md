# 🔑 豆包AI API Key 获取指南

## 📝 具体操作步骤

### 1. 访问火山引擎控制台
🌐 打开浏览器，访问：https://console.volcengine.com/ark/

### 2. 注册/登录账号
- 如果没有账号，先注册一个火山引擎账号
- 已有账号请直接登录

### 3. 进入豆包大模型服务
1. 在控制台首页找到 "人工智能" 分类
2. 点击 "豆包大模型" 或 "ARK"
3. 进入模型服务页面

### 4. 创建推理接入点
1. 点击 "推理接入点" 菜单
2. 点击 "新建推理接入点" 按钮
3. 选择模型：`Doubao-seed-1-6-250615` (或其他可用模型)
4. 设置接入点名称：`XianyuAutoAgent`
5. 确认创建

### 5. 获取API Key
1. 在推理接入点列表中，找到刚创建的接入点
2. 点击 "查看详情" 或 "配置"
3. 在详情页面中找到 "API Key" 部分
4. 复制您的API Key (格式类似: `ak-xxxxxxxxxxxxxxxxxxxxxxxxxx`)

### 6. 更新配置文件
编辑项目中的 `.env` 文件：

```env
# 替换为您复制的真实API Key
ARK_API_KEY=ak-your-actual-api-key-here
```

### 7. 测试连接
```bash
python3 test_doubao.py
```

## 🔍 API Key 格式说明

豆包AI的API Key通常格式为：
- 以 `ak-` 开头
- 后跟一长串字母数字组合
- 总长度约30-40个字符

❌ **错误示例**：`your_ark_api_key_here`
✅ **正确示例**：`ak-1234567890abcdef1234567890abcdef`

## ❓ 常见问题

### Q: 找不到推理接入点菜单？
A: 请确认您已经开通了豆包大模型服务，可能需要实名认证。

### Q: API Key看起来不对？
A: 确保复制完整的API Key，注意不要包含多余的空格或换行符。

### Q: 还是报认证错误？
A: 
1. 检查API Key是否正确复制
2. 确认推理接入点状态正常
3. 验证账户余额充足

## 🎯 完成配置后

1. 运行 `python3 test_doubao.py` 测试连接
2. 看到 ✅ 连接成功后，就可以启动 `python3 main.py`
